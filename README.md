# Hospital_data_analysis

analyzed a hospital dataset and extracted meaningful insights using SQL queries.

📊 What I worked on:

1. Data aggregation using SUM, AVG
   
3. Grouping & filtering datasets
   
5. Identifying trends across departments and hospitals
   
7. Analyzing patient stay durations and expenses
   
Generating monthly and city-wise reports

🔍 Key Insights from the Analysis:

1. Certain departments consistently handled a significantly higher number of patients, indicating higher demand and resource utilization.
   
3. A few hospitals recorded notably higher medical expenses, which could reflect advanced facilities or higher patient inflow.
   
5. The average length of patient stay varied across departments, highlighting differences in treatment complexity.
   
7. City-wise analysis showed uneven distribution of patients, suggesting possible gaps in healthcare accessibility.
   
9. Monthly expense trends revealed fluctuations that could be linked to seasonal or operational factors.

💡 This project strengthened my ability to work with real-world datasets and derive actionable insights using SQL.
